The existing growth monitoring workflow is fragmented, paper-dependent, and consistently delivers delayed, error-prone data to decision-makers.

Quantified System Gaps

How It Works

A single messaging session replaces 3 separate paper and digital workflows. No complex app installation. Minimal additional training required. Designed for low digital literacy environments using simple text or voice-based inputs.

1
AWW sends child data via Messaging Bot Familiar interface, no new app, works on any Android phone with basic data.
2
AI Engine validates in near real-time Flags outliers, missing values, and anthropometric anomalies instantly.
3
Instant feedback to AWW Clear action prompt: normal / MAM / SAM status with referral guidance.
4
Data syncs to cloud automatically No duplicate entry. Dashboard updated. POSHAN integration layer queued.
5
Supervisor and CDPO notified SAM cases trigger automatic alerts upward — reduces routine supervisory burden.

Integration Note — POSHAN Tracker

This system does not replace POSHAN Tracker or any existing MIS. It functions as a near real-time data capture and validation layer that enhances data quality before synchronisation with POSHAN Tracker, reduces duplicate entry burden, and enables intermediate visibility for district and block officers between formal reporting cycles.

WHAT THIS SYSTEM IS NOT

Does not replace POSHAN Tracker: Acts as a complementary validation and visibility layer.
Does not increase reporting burden: Simplifies data capture into a single messaging session.
Does not require new devices: Works on existing smartphones used by AWWs.
Designed to work within existing ICDS systems: Aligned with Poshan 2.0 guidelines.
